O ficheiro template.tex é o ficheiro que deve ser alterado de forma a incluir o conteúdo do relatório.

O ficheiro myrefs.bib é o ficheiro que deve ser alterado de forma a incluir as referências consideradas necessárias.

Para compilar o relatório correr o comando make na pasta onde se encontram os ficheiros.
Em alternativa correr individualmente os comandos que se encontram no ficheiro makefile.

Caso seja alterado o nome do ficheiro, o makefile/comandos devem, naturalmente, ser actualizados em conformidade.
